# Game features
- Player can move the ship by touch left or  right screen.
- Avoid the red bullet, collect the blue one. One charge enough, ship come to Busrt mode.
- Win the fight to collect scrap and upgrade ship.
- 2 Boss - 6 Fight.
- Using partical, glow shader (Unity Bloom + Material), music.
